This small program has 1 single purpose.
Edit/add/remove user key bindings for Openbox on Raspberry Pi OS.
Should work with other OS using openbox.

To use:

Copy the 2 program files to somewhere, run openbox_keybinder.py with python.

It'll list current keybindings and allow modification.

That is all it needs to do.

No guarantee this works on your system.
